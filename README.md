#GradeBook
GradeBook is a digital implementation of traditional paper grade book that commonly used in Ukrainian higher education system. 

##Technology Stack:
*Front End*
* HTML/CSS
* Angular Material
* AngularJS

*Back End*
* RESTful API
* Java EE
* JAX-RS
* Java Persistence API
* JPA Criteria API
* JSON
* MySQL
* Wildfly 8 Application Server
* Maven
